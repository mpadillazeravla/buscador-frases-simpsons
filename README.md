## API Reference

#### UTILIZAMOS LA API DE SIMPSONS

```http
https://thesimpsonsquoteapi.glitch.me/ 
```

#### Get item

```http
GET https://thesimpsonsquoteapi.glitch.me/quotes?count=num

```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `num`      | `string` | **Required**. NUM of item to fetch |
